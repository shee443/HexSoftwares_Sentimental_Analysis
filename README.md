# Sentiment Analysis - Hex Softwares Internship 
# Project Overview

This repository contains the source code for the _Sentiment Analysis_ project developed as part of my internship at _Hex Softwares_.
The objective of this project is to build a Machine Learning model capable of analyzing customer feedback and categorizing it into **Positive**, **Negative**, or **Neutral** sentiments. This tool is designed to help businesses understand customer emotions and make data-driven decisions to improve their products and services.

# Features
- //Data Preprocessing: Cleaning raw, unstructured customer review text (removing special characters, punctuation, and numbers).
- //Text Normalization: Converting text to lowercase to ensure consistency.
- //Stop Word Removal: Filtering out common words that do not contribute to the overall sentiment.
- //Model Training: Utilizing the **Naive Bayes** algorithm to classify text based on the learned patterns from historical data.
- //Business Insight: Transforming raw reviews into actionable business insights.

# Tech Stack
- Language: Python
- Libraries: Scikit-Learn, NLTK, Pandas, NumPy
- Environment: Jupyter Notebook / Google Colab

# Project Structure
- `dataset/`: Contains the raw and processed datasets.
- `notebooks/`: Contains the main Jupyter Notebook with the full implementation pipeline.
- `model/`: Saved model files (if applicable).
